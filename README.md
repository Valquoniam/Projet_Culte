# Projet_Culte


## requirements

- nodejs
- npm
- python >= 3.6

## start the server

```bash
cd culte-server
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python server.py
```

## start the client

```bash
cd client
npm install
npm start
```

## start the game

- open a browser at http://localhost:3000
- click on "start game" [...]
